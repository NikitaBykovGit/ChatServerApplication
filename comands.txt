In order to run project:
1) Pull files
2) Create virtual environment in the project folder
3) Activate the virtual environment
4) Run next commands
    pip install django
    pip install djangorestframework
    pip install markdown
    pip install django-filter
    pip install django-cors-headers
    pip install channels[daphne]
5) If you want use Redis:
    a) Run the command:
        pip install chanel-redis
    b) Uncomment Redis settings in settings.py and change Redis server port
6) Go in folder with file manage.py and run the command python manage.py runserver
